# 🌿 AgroDesign AI Bot
An intelligent Telegram bot that helps users design their gardens based on soil type and lighting conditions.

## ✨ Features
- **Multi-language support:** Automatically detects User's language (Ukrainian or English).
- **Context-aware:** Collects data about soil (sand, clay, etc.) and sun exposure.
- **AI-Powered:** Uses Pollinations.ai (Flux model) to generate photorealistic landscape designs.

## 🛠 Tech Stack
- **Python** (Aiogram 3.x)
- **Pollinations.ai API**
- **In-memory storage** (FSM)

## 🚀 Getting Started
1. Clone the repo.
2. Install requirements: `pip install aiogram aiohttp`.
3. Add your Bot Token and Pollinations API Key.
4. Run `python bot.py`.
